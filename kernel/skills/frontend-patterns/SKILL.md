# Frontend Patterns Skill

When building frontend, refer to **FRONTEND-BANK.md** for baseline patterns.

## Quick Principles

- Semantic HTML (use right elements)
- Accessibility first (works for everyone)
- Performance matters (fast load, smooth UX)
- Mobile-first responsive
- Progressive enhancement

**Key principle:** Good frontend is accessible, fast, and works for everyone.

See FRONTEND-BANK.md in kernel/banks/ for HTML/CSS/JS best practices, React patterns, and performance optimization.
